# DokiTab - 更美观，更轻量的新标签页


> DokiTab~ 一款高度可定制的浏览器新标签页扩展~

![演示](gif/demo.gif)

## ✨ 主要功能

### 1. 极致的交互效果
- **动态背景**：支持自定义图片或视频作为背景，可调节模糊度、遮罩浓度和页面内阴影，打造专属氛围~
- **交互反馈**：本项目大部分按钮都有较为美观的鼠标交互反馈哦~
- **交互特效**：内置多种可选的输入框打字特效和鼠标跟随特效，让交互不再单调。
![特效展示~](gif/fx.gif)

### 2. 深度个性化配置
- **快捷方式列表**：自由添加常用网站快捷方式，支持自定义图标、网格列数和间距。
- ![列表展示~](gif/icons.gif)
- **自定义图标**：图标可以自定义为任何图片（支持动图），支持自动获取图标，不选择图标将使用内置的表情包库，还可以自定义叠加在图标上层的文字。
- ![图标编辑展示~](gif/icon2.gif)
- **大图标**：图标可以自定义为1*1~3*3的大图标样式。
- **侧边栏设置**：通过直观的侧边栏面板，实时调整所有参数。
- ![侧边栏展示~](gif/sidebar.gif)
- **自定义时钟**：高度可定制的时钟组件，支持显示日期、星期、秒数，可调节字体大小、颜色、粗细和数字切换动画。
- ![时钟展示~](gif/clock.gif)
- **搜索栏**：支持多种搜索引擎切换，可调节宽度和透明度，甚至在简洁模式下隐藏。
- ![搜索栏展示~](gif/search.gif)
- **简洁模式**：一键切换界面布局，隐藏干扰元素，助你保持专注，简洁模式样式可自定义。
- **数据持久化**：所有配置自动保存到本地，支持导出配置文件，自定义图标会使用base64打包存储，即使更换了浏览器也可以一键导入~
- ![存储展示~](gif/save.png)
- **Dock 栏**：美观的底部 Dock 栏，放置最常用的应用，支持鼠标悬停放大效果，还可以自定义1*1~1*6的超宽图标。
- ![Dock展示~](gif/dock.gif)


### 3. 猫猫，好耶！
- **猫猫**：内置可爱的猫猫，陪伴你的每一次浏览。
- **自定义设置**：可调节猫猫位置（左/右）、大小、左右晃动，支持自定义猫猫图片（支持动图）。
- **点击语录**：点击猫猫，猫猫会弹出气泡框说话呢，你也可以自定义猫猫语录哦。
- ![猫猫展示~](gif/neko.gif)

### 4. DokiDoki终端
- **终端**：内置一个形似Linux终端的，终端，支持linux命令，不过没有什么用，只是为了好玩w。
- **音乐播放器**：登录终端后，输入music可以打开音乐播放器（需要联网），不过只塞了作者喜欢的音乐呢。
- **彩蛋**：有一个彩蛋可以在终端触发（触发彩蛋后会进入全屏模式，按下F11退出哦）~
- ![终端展示~](gif/termina.png)

## 🚀 快速开始

1. **下载扩展**
   - 前往 [Releases](https://github.com/NekoQiye/DokiTab/releases) 页面下载最新的 `DokiTab_Release_x.x.x.zip` 压缩包。
   - 或者下载源代码自己构建哦。

2. **解压文件**
   - 将下载好的压缩包解压到一个你喜欢的文件夹里（要记住位置呐，以后使用的时候也不要一不小心删掉啦）。

3. **安装到浏览器**
   - **Edge 浏览器**：在地址栏输入 `edge://extensions/` 并回车（或者点击菜单栏的扩展呐）。
   - **Chrome 浏览器**：在地址栏输入 `chrome://extensions/` 并回车（或者点击菜单栏的扩展呐）。
   - 打开右上角的 **"开发者模式" (Developer mode)** 开关。
   - 点击左上角的 **"加载解压缩的扩展" (Load unpacked)** 按钮。
   - 选择刚刚解压好的文件夹，就安装成功了呢，记得启用的说~

## 🔧 自己构建

### 开发环境

1. **克隆项目**
   ```bash
   git clone https://github.com/NekoQiye/DokiTab.git
   cd DokiTab
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发预览**
   ```bash
   npm run dev
   ```

### 构建与安装

1. **构建项目**
   ```bash
   npm run build
   ```
   构建完成后，会在项目根目录生成 `dist` 文件夹。

2. **在浏览器中加载 (Edge/Chrome)**
   - 打开扩展管理页面 (Edge: `edge://extensions/` | Chrome: `chrome://extensions/`)。
   - 开启右上角的 **"开发者模式"**。
   - 点击 **"加载已解压的扩展"** (Load unpacked)。
   - 选择项目目录下的 `dist` 文件夹。


## 📝 说明

- **文件大小限制**：本项目没有设置文件大小限制呢，但是请勿使用分辨率非常非常高的图片和非常大的动图呢，否则可能会导致浏览器性能问题和配置无法正常保存的情况~
- **背景媒体限制**：由于浏览器安全策略，自定义上传的背景图片/视频通常以 Base64 或 Blob 形式存储在本地，请注意文件大小以免超出存储配额。
- **浏览器兼容性**：本项目基于 Manifest V3 开发，完美支持 Microsoft Edge 和 Google Chrome 最新版本。
- **AI使用说明**：本项目部分代码使用AI(Gemini 3 Pro)辅助开发。（不是说AI很会写代码嘛，为什么生成出的代码全部要改一大半QAQ）
