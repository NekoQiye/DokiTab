## 主要功能

### 1. 极致的交互效果

<div align="center">
<img src="gif/fx.gif" width="600" alt="交互特效">
</div>

- **动态背景**：支持自定义图片或视频作为背景，可调节模糊度、遮罩浓度和页面内阴影~
- **交互反馈**：本项目大部分按钮都有较为美观的鼠标交互反馈哦
- **交互特效**：内置多种可选的输入框打字特效和鼠标跟随特效，让交互不再单调~

---

### 2. 深度个性化配置

#### 快捷方式列表
<div align="center">
<img src="gif/icons.gif" width="600" alt="快捷方式列表">
</div>

自由添加常用网站快捷方式，支持自定义图标、网格列数和间距呢

#### 自定义图标
<div align="center">
<img src="gif/icon2.gif" width="600" alt="图标编辑">
</div>

- 图标可以自定义为任何图片（支持动图）
- 支持自动获取图标
- 不选择图标将使用内置的表情包库（50张随机表情包！）
- 可以自定义叠加在图标上层的文字
- 支持 1×1 ~ 3×3 的大图标样式

#### 侧边栏设置
<div align="center">
<img src="gif/sidebar.gif" width="600" alt="侧边栏">
</div>

通过直观的侧边栏面板，可以实时调整所有参数呐

#### 自定义时钟
<div align="center">
<img src="gif/clock.gif" width="600" alt="时钟">
</div>

高度可定制的时钟组件，支持显示日期、星期、秒数，可调节字体大小、颜色、粗细和数字切换动画呢

#### 搜索栏
<div align="center">
<img src="gif/search.gif" width="600" alt="搜索栏">
</div>

支持多种搜索引擎切换，可调节宽度和透明度，甚至在简洁模式下隐藏呐

#### 数据持久化
<div align="center">
<img src="gif/save.png" width="600" alt="数据存储">
</div>

- 所有配置自动保存到本地
- 支持导出配置文件（记得备份呐，别丢了呢，毕竟Edge有可能发疯删扩展呢）
- 自定义图标使用 base64 打包存储
- 一键导入备份文件~
- 项目没有云端上传功能啦，所以作者不会偷你的数据哦，欢迎look源代码的说~

#### Dock 栏
<div align="center">
<img src="gif/dock.gif" width="600" alt="Dock栏">
</div>

美观的底部 Dock 栏，放置最常用的应用，支持鼠标悬停放大效果，还可以自定义 1×1 ~ 1×6 的超宽图标的说

#### 简洁模式

一键切换界面布局，隐藏干扰元素，助你保持专注，简洁模式样式可自定义呐~

## 快速开始

### 1. 下载扩展
前往 [Releases](https://github.com/NekoQiye/DokiTab/releases) 页面下载最新的 `DokiTab_Release_x.x.x.zip` 压缩包，或者下载源代码自己构建哦

### 2. 解压文件
将下载好的压缩包解压到一个你喜欢的文件夹里（要记住位置呐，以后使用的时候也不要一不小心删掉啦w）

### 3. 安装到浏览器

<table>
<tr>
<td width="50%">

**Edge 浏览器：**
- 在地址栏输入 `edge://extensions/` 并回车

</td>
<td width="50%">

**Chrome 浏览器：**
- 在地址栏输入 `chrome://extensions/` 并回车

</td>
</tr>
</table>

**安装步骤：**
1. 打开右上角的 **"开发者模式"** 开关
2. 点击左上角的 **"加载解压缩的扩展"** 按钮
3. 选择刚刚解压好的文件夹
4. 安装成功！记得启用的说~

## 🔧 自己构建

### 开发环境

**1. 克隆项目**
```bash
git clone https://github.com/NekoQiye/DokiTab.git
cd DokiTab
```

**2. 安装依赖**
```bash
npm install
```

**3. 启动开发预览**
```bash
npm run dev
```

### 构建与安装

**1. 构建项目**
```bash
npm run build
```
构建完成后，会在项目根目录生成 `dist` 文件夹呢

**2. 在浏览器中加载**
- 打开扩展管理页面（Edge: `edge://extensions/` | Chrome: `chrome://extensions/`）
- 开启右上角的 **"开发者模式"**
- 点击 **"加载已解压的扩展"**
- 选择项目目录下的 `dist` 文件夹~

## 使用说明

> **重要提示**

- **文件大小限制**：本项目没有设置文件大小限制呢，但是请勿使用分辨率非常非常高的图片和非常大的动图呢，否则可能会导致浏览器性能问题和配置无法正常保存的情况~（补药在传100mb超大GIF给图标惹，求求你啦QAQ）
- **背景媒体限制**：由于浏览器安全策略，自定义上传的背景图片以 Base64 或 Blob 形式存储在本地，请注意文件大小啦~
- **浏览器兼容性**：本项目基于 Manifest V3 开发，完美支持 Microsoft Edge 和 Google Chrome 最新版本呢
- **AI 使用说明**：本项目部分代码使用 AI (Gemini 3 Pro) 辅助开发。（不是说 AI 很会写代码嘛，为什么生成出的代码全部要改一大半 QAQ）
